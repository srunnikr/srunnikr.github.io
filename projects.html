Robocop

Beaglebone, NodeJS based Surveillance Robot

Robocop is Beaglebone based surveillance robot developed at UCSD. It uses iRobot create as the base platform. We have vibration sensors attached to the window and door panes, which detects any vibration or tampering signals when the system is enabled. If it detects someone is trying to break in, it sends a message to the NodeJS web server running on the Robot. It goes to the place of disturbance, takes a photo using the attached Logitech webcam and emails it to the user/administrator. The project was meant to understand and study the inherent complexities of connected sensor systems and robotics.

My contributions include

  • Overall system architecture
  • Complete Embedded software design and development
  • Node JS core server on robot : sensor communication and email API using Python

3D Trilateration and Modelling

3D object modelling using ultrasound trilateration

Designed and developed a 3D trilateration system using ultround sensors and an ARM-Cortex-M4 based embedded processor. Utilized linear and angular information from HC-SR04 ultrasound sensors for accuracy .The algorithm was run on STM32F407VG platform from ST microelectronics. Also developed a custom communication protocol between embedded system and the python based visualization program. The system program is a PyQt based tool that captures, process and plot 3D coordinate data obtained from ARM processor over USB serial interface. The project was nominated for TCS best project award

My contributions include

  • Firmware design and development for the ARM platform
  • Firmware drivers for UART communication as well as sensor interfacing
  • Python based command module for interfacing with the controller and a few modules for 3D plotting
